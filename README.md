# binaryloader-ui

iOS UI components library distributed as a CocoaPods pod.

## Modules

### InsetTextField
UITextField subclass with customizable insets (top, left, bottom, right) via IBInspectable properties.

## Installation

```ruby
source 'https://github.com/binaryloader/cocoapods-specs.git'

pod 'BinaryLoaderUI/InsetTextField'
```
